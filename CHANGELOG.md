## [1.0.0] - 2024-02-19

### Added
- Initial release of EKS Upgrade Planner
- EKS cluster scanning functionality
- Kubernetes resource analysis
- Compatibility checking for addons and APIs
- Upgrade path generation
- Risk assessment capabilities
- Migration planning
- Report generation in Markdown, JSON, and HTML formats
- Docker support
- Complete test suite (19 tests)
- CI/CD pipeline with GitHub Actions
- Cross-platform support (Windows, Linux, macOS)
- Platform-safe CLI symbols
- Comprehensive documentation

### Technical Details
- Python 3.8+ support
- AWS SDK integration (boto3)
- Kubernetes client integration
- YAML-based configuration for compatibility data
- Modern Python packaging (pyproject.toml)
- Black code formatting
